# Coffee Shop Sales Analysis ☕

## Overview

This project focuses on analyzing coffee shop sales data using Microsoft Excel. The dataset was cleaned, organized, and analyzed to identify sales trends, product performance, and business insights. Interactive dashboards and visualizations were created to present key metrics in a clear and user-friendly format.

## Tools Used ⚙️

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Dashboard Visualization

## Key Insights 📊

* Analysis of overall sales performance
* Product-wise sales comparison
* Category-wise revenue analysis
* Customer purchasing trends
* Interactive dashboard for data exploration

## How to Use ?

1. Download or clone this repository.
2. Open the Excel workbook (`.xlsx`) using Microsoft Excel.
3. Enable editing if prompted.
4. Explore the dashboard and worksheets to view the analysis and insights.

## Conclusion 😊

This project demonstrates the use of Microsoft Excel for data analysis and dashboard creation, enabling data-driven decision-making through effective visualization and reporting.
